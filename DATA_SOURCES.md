# Data sources

This project is built from U.S. legislative data, starting from GovInfo Bulkdata XML.

## Primary source
- GovInfo Bulkdata portal: https://www.govinfo.gov/bulkdata/

## Collections used (typical)
- **BILLS** (bill text packages)
- **BILLSTATUS** (bill status/metadata and summaries)

> Note: The full raw XML is not included in this repo. The repo provides derived artifacts (features, cluster cards, metrics, and example rationales).
